# Documentation Images

This directory contains images used in the Kura documentation.

## Adding Images

When creating documentation, add any screenshots, diagrams, or other visual assets in this directory.

## Referencing Images

To reference images in your documentation, use Markdown image syntax with a relative path:

```markdown
![Alt Text](../assets/images/image-name.png)
```
